# Rock Paper Scissors
A project incorporating simple HTML, CSS, and Javascript to create a browser-based Rock Paper Scissors game. Part of the Odin Project cirriculum. 

## Topics Covered
    Dynamic HTML + CSS styling
    Javascript constants, variables, functions
    Best git practices
    Maintaining a GitHub repository
